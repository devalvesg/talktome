# ds/ — Design System em código

Vocabulário comum das telas. **Congelado após o M1** (mudança exige aviso no
Quadro de Alocação).

```
ds/
  tokens/      # re-exports/tipagem dos tokens (CSS vive em src/styles/tokens.css)
  components/  # Btn, Input, Card, Pill, Toast, Modal, Alert, LibrasViewer, ...
```

- **Marco:** M1.
- **Divisão:** Dev A = componentes core · Dev B = componentes de produto.
- **Referência:** [[Design System]] · `docs/design-system-bundle/`.
